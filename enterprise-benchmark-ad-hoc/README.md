# Enterprise Benchmark Ad-Hoc Material

This directory intentionally holds exploratory scripts, one-off reports, local
run outputs, generated files, and old helpers that were used during benchmark
development.

These files are kept so no work is lost, but they are not part of the clean
reproducible benchmark path. They should be reviewed, cleaned up, documented, or
deleted later before any long-lived public branch or release is cut.

The clean benchmark entrypoint directory is `enterprise-benchmark/`.
